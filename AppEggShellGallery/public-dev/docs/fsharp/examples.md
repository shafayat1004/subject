# Where to Find Examples

The best source of worked examples is the running code in this repository.

- **Component usage** — every `LibClient` component has a live page under the **Components** section of
  this gallery, written in pure F#. Read the gallery's own `src/Components/Content/` pages for the exact
  call sites.
- **A full application** — `SuiteTodo` / `AppTodo` is the reference full-stack app (subjects on the
  backend, a pure-F# frontend). See [How To → Where to find examples](./how-to/projects.md).
- **The framework itself** — `LibClient`, `LibRouter`, `LibUi*` are the canonical examples of framework
  conventions (pure-F# `[<Component>]`, styling, theming, accessibility).
