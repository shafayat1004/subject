# Modernization & Current Status

Where the modernization effort stands today, and where it is going. This page is the authoritative
status dashboard. Detail pages are linked from each workstream row.

| Fable.Core | **5.0.0** | |

| Frontend primitives | **`react-native-web` 0.21.2** (web) / **`react-native` 0.86.0** (native) | Off `@chaldal/reactxp` entirely; the ReactXP *name* is also gone — the seam is now `LibClient/src/Rn/` (namespace `Rn`, constructor prefix `Rn.`, `RnPrimitives.fs`). ## Goals status (Goals A through I + Security)

See [Goals & Roadmap](./modernization/goals-and-roadmap.md) for full detail on each goal.

| Goal | Description | Status |
|------|-------------|--------|
| **A** | Retire the render DSL | **Done (product code).** 0 `.render` files under `Lib*`, `ThirdParty`, or `AppEggShellGallery`. 41 remain as compiler test fixtures only under `Meta/AppRenderDslCompiler/compiler.Tests/tests/`. DSL machinery deletion is the remaining closure step. |
| **B** | Fix `eggshell create-app` scaffolding | Not started. |
| **C** | Reduce component verbosity | Incremental, ongoing alongside A. `[<Component>]` + hooks path established; `~16` components still use `TapCapture` shim. |
| **D** | Standardize frontend directory structure | Incremental alongside A/B. `_autogenerated_` removal (post-DSL) is the big simplification. |
| **E** | Speed up frontend build | Partial. Render subprocess stage eliminated by Goal A. Quick wins (precompile cache, parallel type-check) not yet applied. |
| **F** | Platform baseline: .NET 10 + Fable 5 | **Done.** Migrated on branch `modernization/fable5-migration`, now in `modernization/rnw`. Backend TFM bump (net7 to net10 per project) is a separate later step. |
| **G** | PostgreSQL + Orleans 3.7 to 7.x upgrade | Not started. One coordinated workstream; see [Phased Plan](./modernization/phased-plan.md) Phase 3 / later. |
| **H** | ReactXP to react-native-web seam swap | **Done + modernized.** Primitives on RN/RNW, `@chaldal/reactxp` removed, and the ReactXP *name*/seam retired (now `Rn`). Upgraded to React 19 / RN 0.86 / RNW 0.21.2 with the **New Architecture (Fabric)** enabled and verified on device (AppTodo). Remaining: bring gallery + PerformancePlayground onto the RN 0.86 native config; add Reanimated 4. See [ReactXP to RNW](./modernization/reactxp-to-rnw.md). |
| **I** | Frontend render hygiene (key warnings + style leaks) | In progress. Key-warning fix applied at `AppShell/Context/Context.fs`; style-leak rule codified; targeted memoization applied to several components. |
| **Security** | Auth/SQL/crypto hardening | Not started. See [Security Review](./modernization/security-review.md) for the candidate findings table. |
